基本操作：

dev：设备的名称文件

home：普通用户

root:root用户的

bin/sbin:命令的程序文件

etc：配置文件

var:主目录，安装位置

命令与命令/选项/文件/

useradd 名称 //添加用户

pwd:当前所在完整路径 ~代表用户目录 #为当前用户为root $当前用户为普通用户

su 用户名：切换用户

cd 完整路径：切换到指定文件（/为根目录）按TAB

cd .. //返回上一级目录

cd ./:表示当前路径

Ctrl+l：清屏

history:查看历史命令

！数字：执行历史命令的第几条命令

shutdown -h now //立即关机

init 0:立即关机

init 6：立即重启

常用的shell 命令

ls:查看当前目录下面文件

ls /文件名：查看指定文件下的文件

​ 蓝色代表文件夹（目录）

​ 灰色代表普通文件

​ 红色代表压缩文件

​ 绿色代表可执行文件

​ 浅绿色代表链接文件（快捷文件）：硬链接，软连接

ls -l /文件名：当前文件下文件的详细信息（文件类型（-表示普通文件，d表示文件夹，l表示链接文件，.表示隐藏文件），权限，连接文件数量，文件所有者，属组，大小，创建时间，名字）

ls -a表示所有文件包括隐藏文件

ls -lh ：用单位表示文件的大小

ls –help：用法

通配符：

？匹配文件名中的一个字符

“*”表示多个字符

mkdir /文件路径及文件:创建文件

mkdir -p 文件路径及文件

权限三位一组 (第一组为文件所有者，第二位属组权限，第三组是其他用户权限 ，.表示ACL权限) 读（r：权限值为4），写（w 权限值为2），执行(x 权限值为1)

mkdir -m 权限 文件名

rmdir -p 文件：删除空目录

rm -rf 文件：删除文件

cp 原文件路径 复制到的地方:复制文件

cp -R 原文件路径 复制的地方：复制目录

mv 原文件路径 复制的地方：移动文件

touch /文件名：新建文件

chmode 权限值 文件名 //修改文件的权限

​ 字母方式：a:所有用户 u：所有者 g：属组 o：其他用户

​ chmode a = rwx AA

​ chmode u-w,g-wx,o-rwx AA//表示AA文件当前权限为540

head -数字 文件名：查看文件前几行

tail -数字 文件名：查看文件后几行

wc 统计行数

find 范围 查找条件(-name,-type,-user,-size)

grep -选项 条件 名称:查找文件中的内容

​ 选项：-c

man find ：

标准输入：0（键盘）
标准输出：1（屏幕）
错误输出：2（屏幕）
